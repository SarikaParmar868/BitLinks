BitLinks: URL Shortener

Overview:
BitLinks is a fast, privacy-first URL shortener built using Next.js and Tailwind CSS. It allows users to convert long, complex URLs into short, easy-to-share links without requiring login or tracking personal information.

Features:
- Shorten any URL quickly and securely
- Clean, intuitive interface with local custom fonts
- No personal data collection or login required
- Instant copy-to-clipboard for generated links

Setup and Usage:
1. Clone the repository:
   git clone https://github.com/SarikaParmar868/BitLinks.git
2. Navigate to the project folder:
   cd BitLinks
3. Install dependencies:
   npm install
4. Run the development server:
   npm run dev
5. Open your browser at http://localhost:3000 to use the app.

Usage:
Enter a long URL on the homepage and click "Shorten" to generate a short link. Use the "Copy" button to share the link easily.
